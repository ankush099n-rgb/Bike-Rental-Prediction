# 🚲 Bike Rental Prediction System

## 📌 Project Description
This project uses Machine Learning to predict the number of bike rentals based on factors like temperature, humidity, weather, and time.

## 🎯 Objective
To build a regression model that predicts bike rental demand.

## 📊 Dataset
- Source: Kaggle
- Format: CSV

### Features:
- Temperature
- Humidity
- Wind Speed
- Season
- Year
- Month
- Hour

### Target:
- Count of Bike Rentals

## ⚙️ Model
- Linear Regression
- Decision Tree
- Random Forest (Best)

## 🚀 How to Run

Clone the repo:
git clone https://github.com/ankush099n-rgb/Bike-Rental-Prediction.git

Install dependencies:
pip install -r requirements.txt

Run notebook:
jupyter notebook

## 📈 Results
Model gives good prediction accuracy.
