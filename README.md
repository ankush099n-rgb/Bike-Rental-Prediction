# 🚲 Bike Rental Prediction System

## 📌 Project Description
This project uses Machine Learning to predict the number of bike rentals based on various factors such as temperature, humidity, weather conditions, and time.  
It is a regression-based problem where the output is a continuous value (number of rentals).

---

## 🎯 Objective
The goal of this project is to build a predictive model that can estimate bike rental demand, helping in better planning and resource management.

---

## 📊 Dataset
- Source: Kaggle  
- Format: CSV file  
- Type: Structured Dataset  

### 🔹 Features Used:
- Temperature  
- Humidity  
- Wind Speed  
- Season  
- Year  
- Month  
- Hour  

### 🔹 Target Variable:
- **Count of Bike Rentals**

---

## ⚙️ Model Details
- Linear Regression  
- Decision Tree  
- Random Forest (**Best Performing**)
